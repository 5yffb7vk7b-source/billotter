# 🦦 Otterbill

**Free invoice generator that runs entirely in your browser.** No signup, no uploads, no backend — your data never leaves your device.

**Live app → https://otterbill.com/**

## Features

- Professional invoice with logo, line items, tax, discount, partial payments
- 26 currencies with correct locale formatting
- Autosaves to your browser's local storage
- One-click PDF via the browser's print dialog
- Zero network requests with your data — verify it in DevTools

## Pro

A one-time $9 unlock removes the small "Created with Otterbill" line from PDFs. Everything else is free forever.

## Development

It's a static site — no build step. Serve the folder and open it:

```bash
python3 -m http.server 8000
```
